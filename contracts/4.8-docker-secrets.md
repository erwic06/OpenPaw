# Task 4.8 -- Docker and Secrets Configuration

## Objective

Wire Gemini API key and BrowserUse Cloud API key into the Docker configuration. Instantiate `ResearchRunner` in `src/index.ts` at startup.

## Context

The final wiring task for Phase 4. The `gemini_api_key` secret slot already exists in `docker-compose.yml` but is not loaded in `src/index.ts`. The `browseruse_cloud_api_key` secret needs to be added. The `ResearchRunner` from Task 4.7 needs to be instantiated alongside the existing code `SessionRunner`.

Depends on Task 4.7 (ResearchRunner must exist).

## Requirements

- [ ] Load `gemini_api_key` from secrets in `src/index.ts`
- [ ] Add `browseruse_cloud_api_key` secret to `docker-compose.yml`
- [ ] Load `browseruse_cloud_api_key` from secrets in `src/index.ts`
- [ ] Instantiate `ResearchRunner` with all required deps
- [ ] ResearchRunner starts after restart recovery, alongside code SessionRunner
- [ ] Graceful degradation: if Gemini key missing, log warning and skip research runner (same pattern as OpenAI key)
- [ ] Docker build succeeds with updated compose file
- [ ] SIGTERM handler includes ResearchRunner stop

## Interface Contract

**Inputs:** Docker secrets files.

**Outputs:** Running ResearchRunner instance.

**Side Effects:** Docker compose configuration changes.

## Technical Approach

```typescript
// In src/index.ts, after code SessionRunner setup:
const geminiApiKey = secrets.get("gemini_api_key");
const browserUseApiKey = secrets.get("browseruse_cloud_api_key");

if (geminiApiKey) {
  const researchRunner = new ResearchRunner({
    db,
    sendAlert,
    geminiApiKey,
    anthropicApiKey: anthropicApiKey ?? "",
    browserUseDeps: { cloudApiKey: browserUseApiKey ?? "" },
    planPath: PLAN_PATH,
    systemPromptPath: `${REPO_DIR}/agents/researcher/system_prompt.md`,
    reviewerPromptPath: `${REPO_DIR}/agents/researcher-reviewer/system_prompt.md`,
  });
  // ... plan watcher integration for research tasks
} else {
  console.warn("[nanoclaw] gemini_api_key missing — research pipeline disabled");
}
```
