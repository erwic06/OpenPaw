# Task 4.2 -- Gemini Adapter

## Objective

Implement `GeminiAdapter` that implements the `AgentAdapter` interface, enabling Gemini 3.1 Pro sessions for research tasks. Uses the `@google/genai` SDK with streaming responses.

## Context

The existing adapters (`LLMAdapter` for Claude, `CodexAdapter` for OpenAI Codex) follow a common pattern: DI via deps interface, in-memory session state, background promise, token accumulation, cost logging. This adapter mirrors that pattern for Google's Gemini API.

Depends on Task 4.1 (Gemini pricing and "google" provider type).

## Requirements

- [ ] Install `@google/genai` SDK with exact version pin
- [ ] `GeminiAdapter` implements `AgentAdapter` (trigger/status/output/cancel)
- [ ] `GeminiAdapterDeps` interface with DI: `{ db, geminiApiKey, cwd?, genaiFactory? }`
- [ ] `trigger()` starts a Gemini chat session in background, returns session ID
- [ ] Streaming via `generateContentStream()` with token accumulation from usage metadata
- [ ] `getLastActivityMs()` and `waitForCompletion()` for monitoring integration
- [ ] Function calling support: accepts tool declarations, handles function call/response cycle
- [ ] Session tracking: `insertSession` on start, `updateSession` on completion
- [ ] Cost logging via `logUsage` with Gemini model name
- [ ] `cancel()` via AbortController signal
- [ ] Barrel export from `src/agents/index.ts`
- [ ] Unit tests with mocked SDK (genaiFactory DI)

## Interface Contract

**Inputs:** `AgentInput` (taskId, contractPath, systemPromptPath, modelTier, tools, budgetUsd).

**Outputs:** `AgentOutput` (sessionId, terminalState, artifacts, usage, costUsd, error).

**Side Effects:** SQLite session/cost records. Gemini API calls.

## Technical Approach

    // 1. Create GoogleGenAI client
    // 2. Call generateContentStream with system instruction + prompt

For function calling (BrowserUse integration), Gemini uses `tools` parameter with `functionDeclarations`. When the model returns a function call, the adapter executes it via a provided tool handler and sends the result back.
